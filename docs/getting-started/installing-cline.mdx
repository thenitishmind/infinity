### Installation Options

-   **VS Code Marketplace (Recommended):** Fastest method for standard VS Code and Cursor users.
-   **Open VSX Registry:** For VS Code-compatible editors like VSCodium.

### üõ†Ô∏è VS Code Marketplace: Step-by-Step Setup

Follow these steps to get Infinity up and running:

1.  **Open VS Code:** Launch the VS Code application.

    > ‚ö†Ô∏è **Note:** If VS Code shows "Running extensions might...", click "Allow".

2.  **Open Your Infinity Folder:** In VS Code, open the Infinity folder you created in Documents.
3.  **Navigate to Extensions:** Click on the Extensions icon in the Activity Bar on the side of VS Code (`Ctrl + Shift + X` or `Cmd + Shift + X`).
4.  **Search for 'Infinity':** In the Extensions search bar, type `Infinity`.

1. **Install the Extension:** Click the "Install" button next to the Infinity extension.
2. **Open Infinity:**
    - Click the Infinity icon in the Activity Bar.
    - Or, use the command palette (`Ctrl/Cmd + Shift + P`) and type "Infinity: Open In New Tab" for a better view.
3. **Troubleshooting:** If you don't see the Infinity icon, try restarting VS Code.

> ‚úÖ **Pro Tip:** You should see the Infinity chat window appear in your VS Code editor!

### üåê Open VSX Registry

For VS Code-compatible editors without Marketplace access (like VSCodium and Windsurf):

1. Open your editor.
2. Access the Extensions view.
3. Search for "Infinity".
4. Select "Infinity" by saoudrizwan and click **Install**.
5. Reload if prompted.
